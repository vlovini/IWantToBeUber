Introdução

Esta é uma aplicação Java SpringBoot como API de interface web para conexão junto a um banco de dados, possibilitando as opções de select, update, delete e insert.
O banco de dados é desenhado para representar a estrutura básica de um aplicativo de transporte, como o Uber.
A regra de negócio desta empresa presume que a mesma é proprietária dos carros, e os disponibiliza mediante aluguel para que os motoristas efetuem corridas.
As seguintes tabelas constam no banco:

Marcas
Veículos
Condutores
Passageitos
Alugeis
Corridas

A tabela Marcas lista os nomes das marcas de veículos cadastradas.
A tabela Veículos lista os detalhes de cada veículo disponível na empresa, como placa, renavam, modelo e há um FK para marca do veículo.
A tabela Condutores lista as pessoas e seus respectivos dados, cadastradas na plataforma como condutores de veículos da empresa.
A tabela Passageiros lista as pessoas e seus respectivos dados cadastradas na plataforma como passageiros (clientes) da empresa.
A tabela Alugueis lista os alugueis de veículos por parte dos condutores para efetuação das corridas.
A tabela Corridas registra cada serviço prestado, onde são envolvidos um aluguel (que consta condutor e veículo) e um passageiro.

Cenários de Teste

Foram realizados três cenários de teste funcional da aplicação

Cenário 1 - Tabela Marcas

/veiculos/marcas
Método HTTP: Post
Dado que o usuário deva entrar com um dado nome de marca através do corpo de mensagem {“nome”: “NOME_MARCA”}
Esperado: Resposta do servidor 200;
Como validar: Acessar o banco, realizar Select e procurar por marca adicionada.

/veiculos/marcas/{id}
Método HTTP: Get
Dado que o usuário deva entrar com o id de marca cadastrada no sistema através do parâmetro na URL;
Esperado: Resposta do servidor 200; Objeto JSON do tipo marca;
Como validar: Acessar o banco, realizar Select e procurar pela marca retornada.

/veiculos/marcas
Método HTTP: Get
Dado que o usuário deva requisitar para a URL em questão;
Esperado: Resposta do servidor 200; Lista de objetos JSON do tipo marca;
Como validar: Acessar o banco, realizar Select e procurar pela marca retornada.
